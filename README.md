# Sys::Chown

A port of [Perl's File::Chown](https://github.com/perlancar/perl-File-chown
) to Raku, using [Lizmat's Butterfly project](https://modules.raku.org/dist/P5built-ins:cpan:ELIZABETH)

## Installing

The usual

    zef install Sys::Chown
